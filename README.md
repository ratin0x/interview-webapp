# interview-webapp

## Before You Start
This project supplies an `.nvmrc` file for use with `Node Version Manager`

```
nvm use
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run unit tests
```
npm run test:unit
```
